# Carry On para AndesCraft Chile

El binario distribuido en `mods/` es un port de compatibilidad de Carry On para
Minecraft 26.2. Conserva la licencia GNU LGPL v3 del proyecto original.

- Codigo fuente correspondiente: `source/carryon-fabric-26.2-2.10.0-andescraft.1-sources.jar`
- Licencia: `source/CARRYON-LICENSE-LGPL-3.0.txt`
- Proyecto original: https://github.com/Tschipp/CarryOn

El identificador Fabric se mantiene como `grapandgo` solamente para que el
actualizador de AndesCraft pueda reemplazar automaticamente el mod anterior sin
dejar dos implementaciones incompatibles instaladas al mismo tiempo.
